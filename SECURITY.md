# Security Policy

## Supported Versions

Security updates are provided for the latest stable release only.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in `altbrow`, please report it privately using GitHub Private Vulnerability Reporting.

Please include:

- affected version
- environment / OS
- reproduction steps
- impact assessment
- proof of concept (if available)

## Response Process

Reasonable effort will be made to acknowledge reports and investigate them in a timely manner. Fix timelines depend on severity, complexity, and maintainer availability.

## Coordinated Disclosure

Please do not publicly disclose vulnerabilities before a fix or mitigation is available.
