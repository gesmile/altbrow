# Altbrow

An alternative, crawler-like browser that looks beneath the surface of the semantic web. This project is in a very early alpha stage.


## Usage

```

# pre-requirements
python -m venv .venv
source .venv/bin/activate
# Powershell: .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# call
 .\.venv\Scripts\python.exe app\altbrow.py https://tmp.gedankenfalle.de/html5/index.html

```
